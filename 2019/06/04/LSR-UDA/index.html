Learning Smooth Representation for Unsupervised Domain Adaptation

文章提出了一个local smooth discrepancy，局部平滑差异。实验证明这是领域适应上一个有效的 评价体系

文中提到MCD，MCD是一个很好的方法，也很好理解。作者对MCD提到了一点不足：

However, the reason why a decision boundary robust in the source domain becomes invalid in the target domain is not well explained

就是为什么这个decision boundary在target不好，没有解释，本文从一个角度来解释这个问题，

local smooth discrepancy

a local smooth discrepancy

文章从分类器的boundary说起，过拟合问题就是，out-of-distribution样本没有被这个分类boundary正确划分。在UDA（非监督域适应）中，dataset shift加剧这个问题。



直觉上说，把靠近判断域目标域样本投射到离判别域更远的地方会好点。
离decision boundary近的样本会less smooth。
smoothness在这里的意思：

a sample and the samples around it should belong to the same category

$LSD(x, \theta)=D(C(G(x)+r), C(G(x))), \text { where }|r| \leq \epsilon$

r是噪声
sensitive target指的是容易被misclassificated的样本

    $$L S D(x, \theta)=D\left(C\left(G(x)+r_{r a n}\right), C(G(x))\right), \text { where } r_{r a n}=\frac{m}{|m|_{2}}, m \sim N(0,1)$$
    高斯分布的扰动
