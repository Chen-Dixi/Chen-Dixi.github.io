04 二维数组中的查找

题目描述：在一个 n * m 的二维数组中，每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

解题思路：线性查找，时间复杂度$O(n+m)$。空间复杂度$O(1)$

从右上角开始查找，利用从上到下递增，从左到右递增的特点。

8 二叉树的下一个节点

题目描述：给定一个二叉树和其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的指针。

解题思路：

  • 当前结点的右结点不为空的情况很好想，右结点不为空，那中序遍历的下一个结点即使右子树的最左结点；
  • 若右结点为空，没有右子树
    • 当前结点是其父节点的左结点：下个结点就是父结点
    • 当前节点是其父节点的右结点
      • 沿着父结点一直向上遍历，直到找到一个节点是它父节点的左子节点，那么这个父节点就是我们要找的下一个结点

12 矩阵中的路径

题目描述：判断在一个矩阵中是否存在一条包含某字符串所有字符的路径。路径可以从矩阵中的任意一个格子开始，每一步可以在矩阵中向上下左右移动一个格子。如果一条路径经过了矩阵中的某一个格子，则该路径不能再进入该格子。

解题思路：深度优先搜索。

从board上每个点开始进行搜索，上下左右DFS。搜索过一个字符，把他改成’/‘，这样就表示访问过这个位置的字符，回溯后把他恢复。

14.1 剪绳子

题目描述：把一根绳子剪成多段，并且使得每段的长度乘积最大。没规定一段绳子必须减多少段

解题思路：动态规划

  • 边界条件，dp[1]=dp[2]=1，表示长度为2的绳子最大乘积为1；
  • 状态转移：dp[i] = max(dp[i], max((i - j) * j, j * dp[i - j]))


14.2 剪绳子2

题目描述：答案需要取模 1e9+7（1000000007），如计算初始结果为：1000000008，请返回 1。

解题思路：动态规划就不好做了。
贪心最后的结果只包含2和3(当然当总长度为2和3时单独处理), 那么很显然n >= 5时， 3*(n - 3) >= 2 * (n - 2)，因此我们优先拆成3，最后剩余的拆成2。最后的结果一定是由若干个3和1或2个2组成.

16 数值的整数次方

解题思路：快速幂的思路来做，正常情况下，pow(int x, int n)。x是整数，n也是>0的整数，可以很好做。一个快速幂递归代码就可以实现。

二分法思考快速幂问题，$x^{n}=x^{n / 2} \times x^{n / 2}=\left(x^{2}\right)^{n / 2}$。

  • n为偶数时：$x^{n}=\left(x^{2}\right)^{n / / 2}$
  • n为奇数时：$x^{n}=x\left(x^{2}\right)^{n / / 2}$，会多出一项x。
