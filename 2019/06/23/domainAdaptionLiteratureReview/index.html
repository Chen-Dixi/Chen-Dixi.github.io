四 领域自适应理论

  • Analysis of Representations for Domain Adaptation, Ben-David, 2006
    提出了$\mathcal{H} \Delta \mathcal{H} \text { -divergence }$作为传统分布差异的替代衡量式子，克服了从有限样本估计差异的困难。
  • A theory of learning from different domains, Ben-David, ML2010
    提出理论，限制了目标域上的错误期望的范围， in terms of its source error and the source-target divergence

$$
\begin{array}{c}{R_{T}(h) \leq R_{S}(h)+\frac{1}{2} d_{\mathcal{H} \Delta \mathcal{H}}(S, T)+\lambda} \\ {\text { where } d_{\mathcal{H} \Delta \mathcal{H}}(S, T)=2 \sup _{h, h^{\prime} \in \mathcal{H}}\left|\mathbb{E}_{x \sim S} \mathbb{1}\left[h(x) \neq h^{\prime}(x)\right]-\mathbb{E}_{x \sim T} \mathbb{1}\left[h(x) \neq h^{\prime}(x)\right]\right|} \\ {\lambda=\min \left(R_{S}(h)+R_{T}(h)\right)}\end{array}
$$

  • Bridging Theory and Algorithm for Domain Adaptation,Yuchen Zhang,ICML2019

    Our theory can be seamlessly transformed into an adversarial learning algorithm for domain adaptation, successfully bridging the gap between theory and algorithm.

展示了一个领域自适应的理论和算法角度的分析，在理论和算法之间的GAP上建了一座桥。从Rademacher 复杂度上推导一个新的泛化界限

  • On Deep Domain Adaptation: Some Theoretical
    Understandings,Trung Le,arXiv2019

    Abstract：
    The underlying idea of deep domain adaptation is to bridge the gap between source and target domains in a joint space so that a supervised classifier trained on labeled source data can be nicely transferred to the target domain. This idea is certainly intuitive and powerful, however, limited theoretical understandings have been developed to support its underpinning principle. In this paper, we have provided a rigorous framework to explain why it is possible to close the gap of the target and source domains in the joint space. More specifically, we first study the loss incurred when performing transfer learning from the source to the target domain. This provides a theory that explains and generalizes existing work in deep domain adaptation which was mainly empirical. This enables us to further explain why closing the gap in the joint space can directly minimize the loss incurred for transfer learning between the two domains. To our knowledge, this offers the first theoretical result that characterizes a direct bound on the joint space and the gain of transfer learning via deep domain adaptation

域适应其中一个思想方法是，在一个联合子空间上把源域和目标域的差距Gap减小。这样在源域上训练出的分类器也能很好地迁移到目标域。这篇文章提供了严密的框架来解释为什么可以在联合公共空间缩小源域和目标域的差距。更确切地，研究迁移过程中产生的损失，提供了理论来解释和概括现有主要凭经验的深度域适应工作。能够让我们解释，为什么缩小源域和目标域的差距能直接最小化迁移学习的损失loss。

五 方法概览



5.1 概率分布适配法



  • 边缘分布适配
    • 迁移成分分析Transfer Component Analysis
      • 优化目标：
        $\begin{array}{cl}{\min _{\varphi}} & {\operatorname{Dist}\left(\varphi\left(\mathbf{X}_{S}\right), \varphi\left(\mathbf{X}_{T}\right)\right)+\lambda \Omega(\varphi)} \\ {\text { s.t. }} & {\text { constraints on } \varphi\left(\mathbf{X}_{S}\right) \text { and } \varphi\left(\mathbf{X}_{T}\right)}\end{array}$
      • 最大均值差异（Maximum Mean Discrepancy, MMD）
        $\operatorname{Dist}\left(P\left(X_{S}\right), P\left(X_{T}\right)\right)=\left|\frac{1}{n_{S}} \sum_{i=1}^{n_{S}} \Phi\left(x_{S_{i}}\right)-\frac{1}{n_{T}} \sum_{j=1}^{n_{T}} \Phi\left(x_{T_{j}}\right)\right|_{\mathcal{H}}$
    • 迁移成分分析的扩展


  • 条件分布适配

    • Domain Adaptation of Conditional Probability Models via Feature Subsetting

      • 条件随机场+分布适配
      • 优化目标：
        $
        \begin{array}{c}{\operatorname{argmax}_{\mathbf{w}, s} \sum_{(\mathbf{x}, \mathbf{y}) \in D} \sum_{k \in S} w_{k} f_{k}(\mathbf{x}, \mathbf{y})-\log z_{\mathbf{w}}(\mathbf{x})} \\ {\text { such that } \operatorname{dist}\left(\mathcal{D}, \mathcal{D}^{\prime} | S, D, D^{\prime}\right) \leq \epsilon}\end{array}
        $

    • Conditional Transferrable Components (CTC)

      • 定义条件转移成分，对其进行建模

  • 联合分布适配

    • 联合分布适配JDA，（Joint Distribution Adaptation）
      • 直接用TCA，但是加入条件分配
      • 优化目标：
        $
        \begin{aligned} D\left(\mathcal{D}_{s}, \mathcal{D}_{t}\right) \approx & D\left(P\left(\mathbf{x}_{s}\right), P\left(\mathbf{x}_{t}\right)\right) \\ &+D\left(P\left(y_{s} | \mathbf{x}_{s}\right), P\left(y_{t} | \mathbf{x}_{t}\right)\right) \end{aligned}
        $
      • 联合分布适配 结果普遍优于 单独适配边缘分布或条件分布
    • JDA的一些扩展

5.2 特征选择法

从源域和目标域中选择提取共享的特征，建立统一模型

  • Structural Correspondence Learning (SCL) [Blitzer, ECML-06]
    假设源域和目标域有一些特征可以找出来对齐，把这些feature叫做pivot feature.


上下两个日报，经常出现的词频率不同，他们之间有没有什么可以对齐的关系

  • 通常和分布适配进行结合，选择特征通常利用稀疏矩阵。
  • 没有单独非常出色的工作，除了上面的一个。

5.3 子空间学习法

  • 将源域和目标域变换到相同的子空间，然后建立统一的模型
    • 统计特征变换 (Statistical Feature Transformation)
      将源域和目标域的一些统计特征进行变换对齐





5.4 一些的新方法

  • 与深度学习结合
    比如上面的Deep Coral


  • 结合对抗方法
    深度网络加入对抗，把对抗用在子空间学习中，进行特征变换对齐。
    Y. Ganin ,Domain-adversarial neural network, JMLR-16

  • 非Close Set 领域自适应
    当源域和目标域只共享一部分类别时如何迁移
    Kaichao You, Universal Domain Adaptation ,CVPR2019
    Busto, Open set domain adaptation, ICCV2017

  • 多源域领域自适应
    从不止一个源域中迁移到目标域中，也考虑各个源域之间的差异
    Multi-source Unsupervised Domain Adaptation with Category Shift,Ruijia Xu,et al., CVPR2018
