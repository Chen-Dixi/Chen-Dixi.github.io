Dockerfile

Dockerfile是在执行docker build -t 镜像地址/命名空间/镜像名称:tag .时用到的文件，下面是这个学习赛需要用到的Dockerfile。-t代表要打上标签tag

docker build -t xxx/xxx/xxx:1.0 .指令从当前路径下的Dockerfile中构建一个镜像，放在本地。构建完成后，通过docker image ls查看生成的镜像.
